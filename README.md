# React Native Marketplace

### Prerequisites

- npm

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/valentinaless07/react-native-marketplace
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app and scan the QR code in the Expo client app
   ```sh
   expo start
